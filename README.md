# 🚗 RAFTAR X ROAD

**Race. Dodge. Survive.**

A fast-paced neon-inspired car racing game built using **HTML, CSS, and JavaScript** where players dodge enemy cars, collect coins, beat high scores, and survive the endless road challenge.

## 🚀 Live Demo
https://ayush2614.github.io/raftar-x-road/

---

## ✨ Features

✅ Start Screen UI  
✅ Real-time Endless Racing Gameplay  
✅ Smooth Natural Speed Increase  
✅ Enemy Car Collision Detection  
✅ Coin Collection System  
✅ Score Tracking  
✅ High Score Saving (Local Storage)  
✅ Pause / Resume Feature  
✅ Mobile Touch Controls  
✅ Desktop Keyboard Controls  
✅ Game Over Screen  
✅ Restart Option  
✅ Fully Responsive Design  
✅ Neon Futuristic UI  

---

## 🎮 Controls

### Desktop
```txt
⬅ Left Arrow  → Move Left
➡ Right Arrow → Move Right
P             → Pause / Resume
```

### Mobile
```txt
⬅ Left Button
➡ Right Button
⏸ Pause Button
```
